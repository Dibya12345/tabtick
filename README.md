TabiTick is a travel checklist app that helps people avoid forgetting important items like wallets, medicines, tickets, and chargers when leaving home.
The app focuses on the most common problem during travel : _forgetting essentials at the last moment_.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
